This tool was designed to help people understand differences in communication styles between cultures. With this new knowledge, the user will hopefully be able to seamlessly communicate with people all over the world.

Instructions

  1. Select the country you want to learn about. It can be your own country or a different one.
  2. Select a gesture from the list provided
  3. Submit the form
  4. Discover how the gesture is interpreted in the country selected
  5. Go back to change selections, submit again for different results
  • Let's Get Started!
